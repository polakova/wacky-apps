Sorting out your download…

📲 Installing the .APK on Android:

1. Tap the install button above — the .apk downloads.

2. Open it from your notifications / Downloads.

3. If Android warns you, tap Settings → allow from this source, then Install. (It's safe — the app is offline and asks for no internet.)

Prefer not to install? ⬇️ grab the web version instead

📱 iPhone / iPad — make it an app icon:

iPhones can't install .apk files (that's Apple's rule), but this works just as well:

1. The app file opens/downloads above.

2. Open it in Safari → tap Share → Add to Home Screen.

3. It now lives on your home screen like a real app — fully offline. ☕

💻 On a computer: the buttons above give you the Android .apk (for your phone) and the web version (open in any browser).

Tip: on your phone, use “Add to Home Screen” to pin the web version like an app.
